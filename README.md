# Dark Heresy 2E - Continued

An **UNOFFICIAL** system for playing Dark Heresy 2E on [Foundry VTT](https://foundryvtt.com/).

It provides support for **character sheets only**, game content should be drawn from official source books.

The project is being continued under the GPL-3.0 License after the original author deleted their project.

## About This Fork

This repository is a private modification of the actively maintained [Vendare/DarkHeresy2E-FoundryVTT](https://github.com/Vendare/DarkHeresy2E-FoundryVTT) project. It is not intended to replace or supersede that project.

The main focus of this fork is modernizing the system for current Foundry VTT versions and providing first-class Russian localization support.

> [!CAUTION]
> Development work on this fork has been performed exclusively using artificial intelligence. This fork is provided without any guarantee of correctness, reliability, or data safety. Use it entirely at your own risk, keep backups of your Foundry worlds, and prefer the active upstream project if you need a conventionally maintained system.

## Install
1. Go to the setup page and choose **Game Systems**.
2. Click the **Install System** button, and paste in this [manifest link](https://github.com/DroiD16/DarkHeresy2E-FoundryVTT/releases/latest/download/system.json)
3. Create a Game World using the Dark Heresy system.
